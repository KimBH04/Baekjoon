### 문제 설명

<p>희현이랑 주경이는 다항식 게임을 좋아한다.</p>

<p>다항식 게임이란 어떤 특정한 k 값이 주어지면 다항식</p>

<p>\(p(x) = (1+x)(1+x+x^2)(1+x+x^2+x^3)\dots(1+x+\dots+x^{k-1}+x^k)\)</p>

<p>에 대해서 \(x^N\)의 계수를 빨리 말하는 사람이 이기는 게임이다.</p>

<p>희현이가 주경이를 이길 수 있도록 도와주자.</p>

### 입력 

 <p>첫 줄에 테스트 케이스의 숫자 T가 정수로 주어진다.</p>

<p>이어서 매 테스트 케이스마다 한 줄에 걸쳐 정수 k(1 ≤ k ≤ 20), N(0 ≤ N ≤ k(k + 1) / 2)의 값이 공백으로 구분되어 주어진다.</p>

### 출력 

 <p>매 테스트 케이스마다 한 줄에 걸쳐 다항식 \(p(x) = (1+x)(1+x+x^2)(1+x+x^2+x^3)\dots(1+x+\dots+x^{k-1}+x^k)\)의 \(x^N\)항의 계수를 출력한다.</p>
